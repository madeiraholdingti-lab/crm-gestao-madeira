👁️ Visibilidade do atendimento em tempo real

Consultório, empresa, Iza, Mariana — tudo num painel só. Com filtro por número quando quiser separar. Badge em cada conversa indicando de qual número veio. Roteamento automático: médicos/diretores → Iza, pacientes → Mariana.

O que já funciona hoje
Sistema atual
SDR Zap (conversas WA), Task Flow (tarefas), Disparos em Massa, Contatos, Agenda Google Calendar, Relatórios, Gestão de usuários. Stack: React + Supabase + Evolution API multi-instância + n8n.
O que está custando caro de forma anormal
Problema de custo identificado
O Supabase estava consumindo ~$10/dia com 3 usuários. Causas identificadas e parcialmente corrigidas: cron de disparos rodando a cada 1 minuto (corrigido para 15min), webhooks do Evolution API disparando eventos desnecessários em todas as instâncias (corrigido), edge function n8n-inbound-webhook fazendo 8-19 operações no banco por mensagem (parcialmente corrigido). Monitoramento em andamento.
